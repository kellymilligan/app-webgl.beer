webgl.beer

A beery WebGL experiment to render an interactive beer can in the browser. It's coded with THREE.js and uses a can model created by Mihai Panait.

Stuff to add:

  • Improved control of label size, position and appearance

  • Bottle model and labels
  • Range of package shapes and sizes
